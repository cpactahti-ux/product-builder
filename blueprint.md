
# Lotto Number Generator

## Overview

This application generates 5 sets of 6 unique lottery numbers between 1 and 45. It is built using modern web technologies, including Web Components for reusable UI elements.

## Implemented Features

*   **Lottery Number Generation:** Generates 5 sets of 6 unique random numbers.
*   **Custom Element:** Uses a `<lotto-numbers>` custom element to display the generated numbers.
*   **Modern Design:** A clean and simple interface for displaying the numbers.
